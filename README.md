# GitQuest

Transform any GitHub repository into a text adventure RPG.

## Features

- **AI-Powered Storytelling**: Experience unique narratives generated based on your repository's structure and history.
- **Dynamic Music Generation**: Immerse yourself with procedurally generated music that adapts to the game's atmosphere.
- **GitHub Repository Integration**: Play through the history of any public GitHub repository.
- **Text-Based RPG Gameplay**: Engage in classic text adventure mechanics with a modern twist.

## Installation

```bash
git clone https://github.com/YogliB/gitquest.git
cd gitquest
npm install
```

## Usage

To start the development server:

```bash
npm run dev
# or
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser to start your adventure.

## Technologies

- **Frontend**: Vanilla JavaScript, CSS
- **Backend/AI**: Google Gemini (via AI Studio)
